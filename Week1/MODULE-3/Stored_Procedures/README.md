# Exercise 3: Stored Procedures (Module 3)

This folder contains SQL scripts demonstrating stored procedures using control flow (IF/ELSE), CASE, and looping.

## Files
- `stored_procedures_mysql.sql` (MySQL 8+)
- `stored_procedures_postgresql.sql` (PostgreSQL)

## How to run
1. Copy the relevant script into your SQL client (MySQL Workbench / phpMyAdmin for MySQL, psql / pgAdmin for PostgreSQL).
2. Execute top-to-bottom.
3. Review the test calls at the bottom of the script.
